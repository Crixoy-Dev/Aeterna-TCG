# 🎭 HÉROES PRIMORDIALES DE AETERNA

## 1. VALERIUS, EL GUARDIÁN DEL ALBA
* **Frecuencia Primaria:** Blanco (Orden / Protección)
* **Estilo de Juego:** Control defensivo, curación y desgaste. Ideal para jugadores que prefieren dictar el ritmo del combate mediante el bloqueo.

| Atributo | Modo 1v1: Duelo del Nexo | Modo Campos de Batalla |
| :--- | :--- | :--- |
| **Habilidad** | **Escudo de Sinergia** (2 Orbes) | **Bastión de Luz** (Pasivo) |
| **Efecto** | Otorga +2 de Vida a un aliado. Si el aliado ya tiene la vida al máximo, gana **Escudo Divino**. | Al inicio del combate, otorga a tu unidad más a la izquierda: "Cada vez que esta unidad ataca, tus otros aliados ganan +1/+1". |
| **Estrategia** | Mantener bloqueadores con vida alta para frustrar ataques enemigos. | Potenciar una unidad con mucha vida para bufear a toda la mesa constantemente. |

---

## 2. XYLOS, EL TEJE-VACÍO
* **Frecuencia Primaria:** Azul (Manipulación / Tecnología)
* **Estilo de Juego:** Combo y reactividad. Se especializa en robar cartas y gestionar la "Pila" para interrumpir al rival.

| Atributo | Modo 1v1: Duelo del Nexo | Modo Campos de Batalla |
| :--- | :--- | :--- |
| **Habilidad** | **Eco del Nexo** (2 Orbes) | **Algoritmo de Compra** (1 Oro) |
| **Efecto** | Mira las 3 primeras cartas de tu mazo. Añade una a tu mano. Tu próxima **Reacción** este turno cuesta (1) menos. | El siguiente esbirro que compres este turno cuenta como si fuera de todas las Frecuencias (Colores) para los bonus de Sintonía. |
| **Estrategia** | Filtrar el mazo para encontrar la Reacción (Instantáneo) necesaria para anular al oponente. | Completar sinergias de color difíciles mezclando tipos de unidades de forma eficiente. |

---

## 3. IGNACIA, LA LANZA DE CENIZA
* **Frecuencia Primaria:** Rojo (Agresión / Caos)
* **Estilo de Juego:** Agresivo y arriesgado. Utiliza daño directo y la mecánica de Sobrecarga para ganar antes de que el rival se estabilice.

| Atributo | Modo 1v1: Duelo del Nexo | Modo Campos de Batalla |
| :--- | :--- | :--- |
| **Habilidad** | **Combustión Forzada** (3 Orbes) | **Rastros de Pólvora** (Pasivo) |
| **Efecto** | Inflige 2 de daño. Si el objetivo es el Héroe rival, Ignacia sufre **Sobrecarga (1)** pero el daño aumenta a 4. | Después de que un aliado muera en combate, dispara un proyectil que inflige 3 de daño a un enemigo aleatorio. |
| **Estrategia** | Sacrificar el maná del próximo turno para rematar al oponente con ráfagas de daño. | Crear tableros con muchas unidades pequeñas para disparar la pasiva múltiples veces. |

🛠️ Notas de Diseño del Arquitecto
Valerius está diseñado para castigar a los mazos agresivos usando el sistema de Bloqueo Activo. Al curar a sus bloqueadores, obliga al rival a gastar más recursos de los previstos.

Xylos brilla en la fase de Reacciones. Su habilidad le permite buscar respuestas específicas en la pila de hechizos para anular las jugadas del rival.

Ignacia utiliza la mecánica de Sobrecarga para cerrar partidas rápido, sacrificando su economía futura por un pico de daño inmediato.
